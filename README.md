# Indian Curry 2 — Warszawa

Statyczny, responsywny landing restauracji Indian Curry 2. Nie wymaga instalacji zależności — otwórz `index.html` w przeglądarce.

## Zawartość

- przyjazna dla SEO struktura HTML i dane strukturalne `Restaurant`;
- wyraźne ścieżki konwersji: zamówienie, pełne menu i mapa;
- responsywny, dostępny interfejs z obsługą klawiatury i ograniczonych animacji;
- lokalny obraz hero w `assets/`.

Linki do zamówienia prowadzą do aktualnej platformy restauracji.
